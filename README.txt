Онлайн-приложение с серверной и клиентской частью для отправки сообщений самому себе. 
Прекрасное решение для интровертов, которым бы хотелось социализироваться, но не хочется общаться с людьми.
Серверная часть поддерживает прием и отправку сообщений всем подключеннным клиентам. 
Клиентская часть оформлена в SWING, отправляет и принимает сообщения с сервера. 
Есть возможность масштабирования и подключения нескольких клиентов, однако это противоречит философии приложения.
